# 焚诀：止跌下影线 + 放量突破均线 + 右侧确认

## 核心思想

焚诀用于捕捉一类趋势反转：股价长期横盘或回落后，先出现长下影线确认下方承接，再于 1-3 个交易日内放量站上短中期均线，随后通过右侧趋势确认过滤假突破。

## 三段式规则

1. 止跌预警：长下影占全天振幅不低于 45%，收盘回到日内中位以上，低点接近 40 日箱体低位。
2. 启动确认：止跌后 1-3 日内放量不低于 20 日均量 1.5 倍，收盘至少站上 MA5/MA10/MA20/MA30 中 3 条，且收在日内强势区。
3. 右侧确认：突破后 20 个交易日内价格继续抬升，且后续收盘能站上 MA60，回踩不破启动阳线中位或 MA20 为佳。

## 买卖纪律

- 长下影线本身不是买点，只是预警。
- 买点在放量突破确认 K 线或其后缩量回踩不破时。
- 若跌回启动阳线中位或 MA20，信号降级。
- 若突破后持续站上 MA60，短中期均线转多头，视为右侧趋势确认。

```bash
./apps/TradingAgents-astock/.venv/bin/python -c 'from libs.stock_data.reversal import scan_reversal_signals; print(scan_reversal_signals(["002475"], "2026-06-05", start_date="2025-12-01", refresh=True).to_string(index=False))'
```

拉取市场股票池，排除 `30`、`68` 开头股票，采集半年日线，再筛选回撤：

```bash
./apps/TradingAgents-astock/.venv/bin/python -c 'from libs.stock_data.universe import run_market_reversal_scan; print(run_market_reversal_scan("2026-06-05", start_date="2025-12-01").to_string(index=False))'
```

## 半年回撤验证口径

- 使用新股票数据资产层 `collect_stock_snapshot` 拉取半年日线。
- 用 `daily_qfq.csv` 判定形态和均线。
- 用 `daily_raw.csv` 做后续收益与最大回撤回测。
- 默认持有窗口为 20 个交易日。
